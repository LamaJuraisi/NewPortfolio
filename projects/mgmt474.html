MGMT 474 - Stock Market Analysis

Exploring stock performance for Nike, Adidas, and Puma

Project Insights

Our data-driven analysis provided actionable insights for portfolio diversification. By evaluating market trends of Nike, Adidas, and Puma, we identified predictive models with strong accuracy, optimizing decision-making for financial forecasting.

Did you know? Sports stocks like Nike, Adidas, and Puma often move together, making them an ideal combination for portfolio diversification strategies!

Tools & Techniques

  • R Programming: Used extensively for data analysis, visualizations, and building regression and machine learning models.
  • Regression Analysis: Assessed linear relationships between stock prices for portfolio forecasting.
  • KNN Algorithm: Applied for short-term stock price predictions with low RMSE.
  • Naive Bayes Classifier: Evaluated stock price distributions, albeit with moderate accuracy.

Process Summary

  • Collected stock data from reliable sources, including closing prices, volatility, and trading volumes.
  • Conducted regression and correlation analyses for pattern identification.
  • Built predictive models using Random Forest, KNN, and Naive Bayes algorithms.
  • Evaluated model performance metrics for actionable insights.

Key Code and Results

Code: Observations


# Plotting closing prices
library(ggplot2)
ggplot(data = stock_data, aes(x = Date)) +
  geom_line(aes(y = NKE_Close, color = "Nike")) +
  geom_line(aes(y = ADS_DE_Close, color = "Adidas")) +
  geom_line(aes(y = PUM_DE_Close, color = "Puma")) +
  labs(title = "Closing Prices Over Time for Nike, Adidas, and Puma", 
       x = "Date", y = "Closing Price") +
  theme_minimal()

Interpretation: Nike exhibits higher price volatility and steady growth compared to Adidas and Puma. Strong positive correlation among the three stocks suggests shared market trends.

Results

Linear Regression: Revealed significant relationships between Adidas and Nike stock prices (R² = 0.72).

KNN: Provided reliable short-term predictions for Nike with low RMSE.

Random Forest: Outperformed other models in accuracy, validating its use for predictive stock analysis.

Naive Bayes: Moderate accuracy (45%) indicated limited utility for stock forecasting.
