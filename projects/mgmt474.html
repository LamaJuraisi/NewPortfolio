MGMT 474 - Stock Market Analysis

Exploring stock performance for Nike, Adidas, and Puma

Objective

Analyze the stock performance of Nike, Adidas, and Puma to gain insights into market trends and identify top-performing predictive models for stock price forecasting.

Tools & Techniques

  • R Programming
  • Correlation Analysis
  • Random Forest
  • Naive Bayes, K-Nearest Neighbors

Process Summary

  • Collected stock data for Nike, Adidas, and Puma from reliable sources.
  • Performed regression and correlation analyses to identify patterns and trends.
  • Evaluated predictive models, including Random Forest and Naive Bayes.

Key Code and Results

# Example R Code for Correlation Analysis
