MGMT 388 - Coffee Trade Analysis

Diving deep into global coffee trends using data analytics and predictive modeling

Objective

Analyze global coffee trade trends (2002–2023), leveraging data and machine learning to uncover insights into import trends, production patterns, and consumer behavior.

Regression Analysis

We developed a regression model to predict coffee imports to the US using factors such as Arabica production, domestic consumption, and time.


# Regression Analysis: Predicting Coffee Imports
us_data = data[data['Country'] == 'United States'].drop(columns='Country')
us_data = us_data.rename(columns={
    'Bean Imports': 'Bean_Imports',
    'Domestic Consumption': 'Domestic_Consumption',
    'Arabica Production': 'Arabica_Production',
    'Beginning Stocks': 'Beginning_Stocks'
})

formula = 'Bean_Imports ~ Arabica_Production + Year + Domestic_Consumption'
import_model = smf.ols(formula, data=us_data)
result = import_model.fit()

result.summary()

Insights: The model demonstrated a strong fit with an R-squared value of 0.86. Key predictors like Arabica Production, Domestic Consumption, and Year showed significant impact on coffee imports.
