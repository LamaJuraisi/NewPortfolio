MGMT 388 - Coffee Trade Analysis

A technical exploration of coffee trade data, predictive modeling, and insights

Objective

The project explores coffee trade data from 2002 to 2023, focusing on trends, predictive modeling, and actionable insights.

Process Summary

  • Analyzed global coffee trade data.
  • Explored seasonal trends and country-specific contributions.
  • Applied regression models to predict coffee imports.
  • Clustered data for advanced insights.

Project Insights

Significant market patterns revealed the interconnectedness of production and consumption trends globally.

Did you know? Coffee imports in the US increased by 45% from 2002 to 2023, reflecting evolving consumer preferences.

Tools & Techniques

  • Python: Libraries like `pandas`, `seaborn`, and `statsmodels` for analysis and modeling.
  • Visualizations: Used `matplotlib` and `plotly` for dynamic and interactive visualizations.

Key Code Snippets

Regression Analysis


# Linear Regression for Predicting Imports
import statsmodels.api as sm

X = us_data[['Arabica_Production', 'Year', 'Domestic_Consumption']]
y = us_data['Bean_Imports']
X = sm.add_constant(X)
model = sm.OLS(y, X).fit()
print(model.summary())

Interpretation: The regression model showed an R-squared of 0.86, indicating strong predictive power. Key predictors were statistically significant at the 5% level.

Time-Series Trends


# Seasonal Decomposition of Coffee Imports
from statsmodels.tsa.seasonal import seasonal_decompose

decomposition = seasonal_decompose(us_data['Bean_Imports'], model='additive', period=12)

Results

  • Regression Analysis: Explained 86% of variance in coffee imports using key predictors.
  • Time-Series Analysis: Seasonal trends identified with increased imports during winter months.
  • Cluster Analysis: Segmented countries into high, medium, and low import categories.
