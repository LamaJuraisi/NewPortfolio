Regression Analysis: US Coffee Imports

Preparing Data for Regression


# Filter data for US from 2002 onwards
us_data = data[data['Country'] == 'United States']
us_data = us_data[us_data['Year'] >= 2002]

# Rename columns for easy referencing
us_data = us_data.rename(columns={
    'Bean Imports': 'Bean_Imports',
    'Domestic Consumption': 'Domestic_Consumption',
    'Arabica Production': 'Arabica_Production',
    'Beginning Stocks': 'Beginning_Stocks'
})

# Display the cleaned data
display(us_data.head())

Domestic Consumption Trends


# Visualize domestic consumption over time
import seaborn as sns
import matplotlib.pyplot as plt

sns.scatterplot(data=us_data, x='Year', y='Domestic_Consumption')
plt.title('US Domestic Consumption Over Time')
plt.ylabel('Consumption (thousands of bags)')
plt.show()

Observation: Domestic consumption has shown a consistent upward trend over time.

Regression Model


# Predicting US imports using regression
import statsmodels.formula.api as smf

formula = 'Bean_Imports ~ Arabica_Production + Year + Domestic_Consumption'
model = smf.ols(formula, data=us_data)
result = model.fit()

print(result.summary())

Key Findings:

  • R-squared: 0.86, indicating the model explains 86% of the variance in imports.
  • Significance: All predictors are significant at the 5% level, with p-values of 0.01, 0.02, and 0.00.
