MGMT 388 - Coffee Trade Analysis

A technical exploration of coffee trade data, predictive modeling, and insights

Objective

This project is an analytical dive into the world of coffee trade. Using data from 2002 to 2023, we explored hidden trends and created predictive models to answer critical questions: How do imports evolve over time? What drives these shifts? From uncovering seasonal spikes to segmenting countries based on trade behavior, this journey highlights the blend of numbers and stories—one dataset at a time.

Process Summary

  • Collected and cleaned coffee trade data from 2002 to 2023.
  • Built regression models to predict US coffee imports.
  • Conducted cluster analysis to segment countries by import behavior.
  • Visualized trends across coffee types and seasonal consumption.

Key Code Snippets

Regression Analysis: Predicting Imports


# Linear Regression for Predicting Imports
import statsmodels.api as sm

X = us_data[['Arabica_Production', 'Year', 'Domestic_Consumption']]
y = us_data['Bean_Imports']
X = sm.add_constant(X)
model = sm.OLS(y, X).fit()
print(model.summary())

Insights: The regression model explained 86% of the variance in coffee imports. Key predictors were statistically significant and provide actionable insights for supply chain optimization.

Results

  • Regression Analysis: Explained 86% of variance in coffee imports using key predictors.
